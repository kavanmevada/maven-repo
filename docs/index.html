Gradle / Maven dependency

At the moment we do not have a publishing mechanism to a maven repository so the easiest way to add the library to your app is via a GitLab Dependency

repositories { ... maven { url "https://github.com/kavanmevada/maven-repo/raw/master" } maven { url "https://gitlab.com/kavanmevada/maven-repo/raw/master" } } dependencies { ... implementation 'elementx.playback:playback:1.0.0' }
